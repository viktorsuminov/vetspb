После скачивания проекта нужно создать виртуальное окружение, активировать его, установить зависимости и можно запускать.

Вирутальное окружение должно быть с версией python от 3.10 до 3.12, если у вас версия выше - потребуется обновить библиотеки перед запуском.
# Создание виртуального окружения
python -3.12 -m venv venv

# Активация (Windows)
venv\Scripts\activate

# Активация (Linux/macOS)
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python run.py
